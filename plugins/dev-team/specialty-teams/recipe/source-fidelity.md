## Worker Focus
Every requirement traceable to actual code behavior. No invented requirements for functionality the code doesn't have. Code quirks and workarounds documented in Design Decisions. Missing functionality marked NEEDS REVIEW, not fabricated.

## Verify
Requirements map to observable code; no fabricated capabilities; workarounds documented; gaps marked NEEDS REVIEW not invented.
