## Worker Focus
Identify OS framework imports, SDK usage, system service calls (keychain, filesystem, network), hardware APIs (camera, GPS, bluetooth), and third-party library dependencies. Map each to the code units that use them.

## Verify
All external framework/SDK imports catalogued; system service access points identified; hardware API usage documented per code unit.
