## Worker Focus
RFC 2119 keywords (MUST/SHOULD/MAY) used correctly per their semantics. Each requirement independently testable. Requirements describe behavior not implementation. Specific numeric or string values where applicable rather than vague qualifiers.

## Verify
Every requirement uses RFC 2119 keywords; no vague or untestable requirements; no implementation-specific language; specific numeric values where applicable.
